## 🧾 Sistema de Stock y Punto de Venta (Lola)

Aplicación web desarrollada para gestionar productos, stock, ventas y proveedores.  
Pensada para pequeños comercios que necesitan una solución simple y eficiente.

---

## 🚀 Funcionalidades

- ✔ Alta, baja y modificación de productos
- ✔ Gestión de categorías y proveedores
- ✔ Control de stock en tiempo real
- ✔ Registro de ventas
- ✔ Búsqueda por nombre, proveedor o código de barras
- ✔ Interfaz tipo punto de venta (POS)
- ✔ Cálculo automático de totales

---

## 🛠️ Tecnologías utilizadas

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- JavaScript
